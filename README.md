## 三. 核心功能 (Features)

### 1. 用户：

#### (1). 学生用户:

- 申请功能	向教师申请空教室。
- 预约查询功能    查看用户自己的预约情况。
- 课程查询功能    查找用户自己的课程情况。
- 修改/删除功能    对自己的预约进行删改。
- (可选)签到功能。

#### (2). 教师用户(普通):

- 预约功能	预约空教室。
- 审批功能    审批学生的申请。
- 预约查询功能    查看用户自己的预约情况。
- 课程查询功能    查找用户自己的课程情况。
- 修改/删除功能    对自己的预约进行删改。
- (可选)签到功能。

#### (3). 管理员用户:

- 添加教室功能    新建教学楼，教室等信息。
- 修改教室功能    调整教室状态如维修。
- 添加用户功能    创建学生或教师用户(未激活)。
- 修改用户信息    特定情况下修改用户信息。

### 2. 登录注册：

#### (1). 初始化账号（管理员）：

- 批量注册学生账号。（注：根据身份证号，但无数据，具体方法待定。）
- 添加教师账号。

#### (2). 激活账号：

- 登录初始化信息，完善账号数据。
- 设置安全问题（无法自主更改）。
- 生成复原码。（注：字符串，用于忘记密码时账号登录，需手动保存。）

#### (3). 修改密码：

- 登录时使用复原码，强制失效原密码，并失效使用后的复原码，在重新设定密码后，生成新的复原码。
- 登录后输入密码并通过安全问题生成新的复原码，旧复原码失效。
- 登录后，通过复原码修改密码。
- 多次使用复原码后只能通过向管理员账号发出申请修改密码。
